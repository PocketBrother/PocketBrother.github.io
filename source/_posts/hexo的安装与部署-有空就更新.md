## 创建一篇新文章
```
hexo new "文件名"
```
## 本地启动hexo
```
hexo server
```
## 提交到github上
### 第一步,把本地所有的文件都加入到暂存区
```
git add .
```
### 第二步,把暂存区所有的文件都提交到版本库
```
git commit -m '这里随意填写注释'
```
### 第三步,把本地项目所在的的分支与远程的分支进行合并
```
git push 分支名
```
## 第四步,把hexo部署到线上
```
hexo deploy
```
